# Инструкция для работы с Markdown

## Выделение текста

Чтобы выделить текст курсивом, 
необходимо обрамить его звезздочккми (*).
Например *вот так*.

Чтобы выделить текст полужирным, необходимо обрамить его двойными звездочками (**).
Например **вот так**.
